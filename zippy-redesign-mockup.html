What's changing (structure + proportions only)

Same data, cleaner layout. No new pdf upload, no source pinning — the sources behaviour stays exactly as it is today.

Header slimmed down — logo, name, subtitle in a tight row; "New chat" button moved up top so the sessions list is only for switching.
SESSIONS header row with icons — refresh, search, filter, collapse. Gives a clean divider between history and the active thread.
Session rows get "Completed in Xs" + relative time on the right.
Sessions list capped at ~180px so the thread gets most of the panel height (not the session list).
Composer gets its own rounded box with the keyboard hint inline — more breathing room than the current flat input.
Bottom status chip — "Grounded in N files · Last synced Xm ago" — tiny, always visible, confirms the index is fresh.

Everything stays on your current light palette. Purple stays as Zippy's accent. Same sources behaviour as today.
